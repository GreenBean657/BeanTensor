# BeanTensor
BeanTensor is a C++ library built for HIP, CUDA, and CPU (AVX512 accelerated!) inferencing.
Other libraries have patches for support like this, but it's often buggy or falls apart in high dimensions.
BeanTensor is built on the prinicple of GROUND UP support. 


# Usage
The library is under construction, estimated for completion in December 2026.
Usage is at your own risk, and the API is unlikely to function fully or at all.
